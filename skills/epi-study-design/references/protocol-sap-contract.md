# PROTOCOL 与 SAP 字段合同

## PROTOCOL

- 研究背景与可核验依据
- 研究问题、目标和假设
- 设计、场景、数据来源、研究时间窗与时间零点
- 目标人群、纳入排除、暴露或干预、比较策略
- 主要与次要结局、测量时点、判定来源
- estimand 与中间事件处理
- 偏倚、混杂、缺失和可行性控制
- 伦理、注册、数据治理与适用报告规范
- 版本、冻结状态和确认人

## SAP

- 分析人群与样本流
- 变量编码、派生规则和有序水平
- 主要 estimand、效应量和置信区间
- 主要模型、协变量选择依据和模型诊断
- 缺失数据、异常值、多重性与信息泄漏控制
- 样本量、事件数或精度依据及全部假设
- 预设亚组、交互、敏感性和稳健性分析
- 随机过程、验证方案、软件环境与复现入口
- 方案偏离记录和探索性分析边界

字段不适用时说明原因，不用空白或猜测值填充。研究设计审查以概念完整和定义可执行为准，不以模板字段数量为目标。
