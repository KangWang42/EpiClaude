>本次介绍此R包的第一个函数如何批量修改字符串

### R包安装
```r
install.packages("ggtools_1.0.tar.gz",repos = NULL)
library(tidyverse)
library(ggtools)
```
### 读入映射文件
```
read_tsv("ID.txt",col_names = F)
```
第一列为原始的名称，第二列为新的序列名称；注意顺序即可。
![](https://files.mdnice.com/user/7338/75eec212-4c53-4e7b-b7fd-a8228ac2bac0.png)
```
replace_fasta_ids(mapping_file, fasta_file, output_file)
```
>该函数使用三个参数:   
mapping_file: 映射文件   
fasta_file: 原始名称的fasta序列文件	  
output_file: 替换名称后的fasta序列文件


### 替换fasta序列名称
```
ggtools::replace_fasta_ids("ID.txt","HvOSCA.pep.fa","HvOSCA.pep.rename.fa")
```
上面介绍的是针对fasta文件来进行的数据替换，那么该函数进行进一步的修改也可对数据框中的字符串进行批量替换。
### 替换数据框字符
```
replace_chars(file, char_pairs, output_file)
```
>该函数使用三个参数:   
file：要替换的文件  
char_pairs：字符对的数据帧，包含两列："old"（需要替换的旧字符）和"new"（新字符）
output_file：替换后的文件
### 使用案例
若要替换的数据较小可以自己构建数据框进行替换，若有较多的内容需要替换可以读入本地的映射文件来进行新旧名称的替换。
```r
ggtools::replace_chars("original_file.txt",
                       data.frame(old = c("old_char1", "old_char2", "old_char3"),
                                  new = c("new_char1", "new_char2", "new_char3")),
                       "replaced_file.txt")
```
### 实际应用
```r
id <- read_tsv("ID.txt",col_names = F)
replace_chars("HvOSCA.pep.fa",id,"rename.fa")
```
