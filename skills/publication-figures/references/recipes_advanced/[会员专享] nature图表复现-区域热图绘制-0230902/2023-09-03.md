### 图形解读
>ggmagnify包默认的函数可以很轻松的对选定的区域进行缩放，但是绘制此图难点在于如何只在缩放图中出现需要的Y轴文本，小编无意中想的了一个窍门解决了此问题，有需要的继续往下阅读。

### 加载R包
```r
install.packages("ggmagnify", repos = c("https://hughjonesd.r-universe.dev", 
                                        "https://cloud.r-project.org"))
library(tidyverse)
library(readxl)
library(magrittr)
library(ggtree)
library(ggmagnify)
library(grid)
library(patchwork)
```
### 导入数据
```r
df <- read_excel("F1.xlsx", sheet = "Fig 1c KEGG module") %>% 
  column_to_rownames(var="...1")
```
### 数据清洗
```r
df2 <- df %>% rownames_to_column(var="ID") %>% 
  pivot_longer(-1) %>% 
  set_colnames(c("ID","name","value"))
```
### 聚类分析
```r
hc <- hclust(dist(t(df))) %>% ggtree() + layout_dendrogram()+
  theme_void()
```




### 绘制热图
>在此只展示需要缩放区域的Y轴文本名称，但是这样设置在主图与附图中均会展示，所以后面还需代码调整
```r
heatmap <- df2 %>% 
  ggplot(aes(name,ID,fill=value))+
  geom_tile()+
  labs(x=NULL,y=NULL)+
  scale_y_discrete(expand=c(0,0),position="left",breaks=c("M00001","M00002","M00015",
                                                          "M00028","M00032","M00038",
                                                          "M00044","M00060"))+
  scale_x_discrete(expand=c(0,0))+
  scale_fill_gradientn(colours = rev(RColorBrewer::brewer.pal(11,"RdBu")))+
  coord_cartesian(clip = "off") + 
  theme(axis.text.x=element_blank(),
        axis.ticks = element_blank(),
        legend.title = element_blank(),
        legend.background = element_blank(),
        legend.text = element_text(size=8,color="black"),
        legend.position =c(1.2,0.8),
        plot.margin = ggplot2::margin(10,150, 10,1))
```
### 区域放大热图
```r
from3 <-  c(2,20,1,8)
to3 <- c(150,210,10,25)

p <- heatmap + 
  geom_magnify(from = from3, to = to3, axes = "y") +
  theme(axis.text.y=element_text(color="white"))  # 这里的的代码意为将主图的文本修改为白色
```
>那么可以这么理解若同时定义两个**axis.text.y**则前一个定义的为放大区域图的文本属性，后一个则是主图的文本属性，怪哉。
