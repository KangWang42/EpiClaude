### 案例1 绘制直方图
```C
histogram <- gss_cat %>% 
  nest(data=-marital) %>% 
  mutate(histogram=pmap(
    .l=list(marital,data),
    .f=\(marital,data){
      ggplot(data,aes(x=tvhours))+
        geom_histogram(binwidth = 1)+
        labs(title=marital)
    }
  )
)
```
### 代码过程解读
#### 步骤1
>gss_cat按照marital字段进行嵌套（nest）。这会将除了marital字段之外的所有字段组合成一个名为data的嵌套数据框。

#### 步骤2
>使用`mutate`函数在原始数据框中创建一个名为histogram的新列，这个新列会包含每个婚姻状况（marital）对应的直方图。

>`pmap`函数用于对输入的列表（marital和data）进行并行映射操作。该函数的第一个参数是一个列表，它包含了marital字段和对应的嵌套数据。  
第二个参数是一个匿名函数，它定义了如何根据输入的marital和data生成直方图。

>在这个匿名函数中，使用了ggplot2包函数来创建直方图。  
`ggplot`函数创建了一个空白的图形对象  
`aes(x=tvhours)`设置了横坐标为tvhours  
`geom_histogram`函数用于在图形对象上添加直方图  
`binwidth = 1`表示直方图的每个柱子的宽度为1  
`labs(title=marital)`用于设置直方图的标题为对应的婚姻状况

```r
color_scale <- c("Adelie" = "darkorange", "Chinstrap" = "purple","Gentoo" = "cyan4")
```
```C
species <- Species %>% 
  nest(data = -name) %>% 
  mutate(name = pmap(
    .l = list(name, data),
    .f = \(name, data) {
      ggplot(data, aes(species, value, color = species)) +
        geom_violin(aes(fill = species), alpha = 0.40) +
        geom_beeswarm() +
        scale_color_manual(values = color_scale) +
        scale_fill_manual(values = color_scale) +
        theme_bw() +
        labs(x=NULL,y=NULL)+
        theme(legend.position = "none",
              axis.text = element_text(color = "black",face="bold"),
              plot.margin = margin(5, 5, 5, 5),
              panel.spacing = unit(0.2, "lines"),
              panel.background = element_rect(color = "black", fill = NA))
    }
  )
  )
```
