# 论文各部分模板

## 摘要 (Abstract)

```
目的：探讨 [暴露因素] 与 [结局] 的关联。

方法：采用 [研究设计]，纳入 [N] 名 [研究对象]。使用 [统计方法] 分析 [暴露] 与 [结局] 的关系，调整 [协变量]。

结果：[描述主要发现]。多变量分析显示，[暴露] 与 [结局] 显著相关（OR/HR = X.XX，95% CI: X.XX-X.XX，P = X.XXX）。

结论：[一句话结论]。
```

---

## 引言 (Introduction)

### 第1段：研究背景

```
[疾病/问题] 是全球/我国重要的公共卫生问题。据统计，... [待补充引用]。
[疾病] 的发生发展受到多种因素的影响，包括 ... [待补充引用]。
```

### 第2段：既往研究

```
既往研究表明，[暴露因素] 与 [结局] 存在关联。[作者] 等发现 ... [待补充引用]。
然而，既往研究存在以下不足：(1) ...; (2) ...; (3) ...。
```

### 第3段：研究目的

```
因此，本研究旨在利用 [数据来源]，探讨 [暴露] 与 [结局] 的关联，
为 [实际意义] 提供科学依据。
```

---

## 方法 (Methods)

### 研究设计

```
本研究为 [队列研究/病例对照研究/横断面研究]。
```

### 研究对象

```
研究对象来源于 [数据库/医院/社区]，纳入标准：(1) ...; (2) ...。
排除标准：(1) ...; (2) ...。
最终纳入 [N] 名研究对象。
```

### 变量定义

```
暴露变量：[定义]。

结局变量：[定义]。

协变量：包括年龄、性别、[其他变量]。
```

### 统计分析

```
使用 R 软件 (版本 X.X.X) 进行统计分析。
连续变量以均数 ± 标准差或中位数（四分位距）描述；
分类变量以频数（百分比）描述。
采用 [t检验/卡方检验] 比较组间差异。
采用 [逻辑回归/Cox回归] 分析 [暴露] 与 [结局] 的关联，
模型1调整 [协变量1]，模型2进一步调整 [协变量2]。
P < 0.05 认为差异有统计学意义。
```

---

## 结果 (Results)

### 描述性统计

```
共纳入 [N] 名研究对象，其中 [结局组] [n] 名 (XX.X%)。
两组在 [变量] 方面存在显著差异 (P < 0.05)（表1）。
```

### 主分析

```
单因素分析显示，[暴露] 与 [结局] 显著相关 (OR = X.XX, 95% CI: X.XX-X.XX)。
调整 [协变量] 后，关联仍然显著 (OR = X.XX, 95% CI: X.XX-X.XX, P = X.XXX)（表2）。

[可插入森林图描述]
```

### 敏感性分析

```
敏感性分析结果与主分析一致，表明结果稳健。
```

---

## 讨论 (Discussion)

### 第1段：主要发现

```
本研究发现，[暴露] 与 [结局] 存在 [正向/负向] 关联。
与既往研究 [一致/不一致] [待补充引用]。
```

### 第2段：机制探讨

```
可能的机制包括：... [待补充引用]
```

### 第3段：优势与局限

```
本研究的优势包括：(1) [样本量大]; (2) [调整了多种混杂因素]。

局限性包括：(1) [横断面设计无法确定因果关系]; (2) [可能存在未测量的混杂]；
(3) [自报偏倚]。
```

### 第4段：结论

```
综上所述，[一句话结论]。未来研究应 [建议]。
```

## Word 导出代码

```r
library(officer)
library(flextable)

# 创建文档
doc <- read_docx() |>
  # 标题
  body_add_par("论文标题", style = "Title") |>
  
  # 摘要
  body_add_par("摘要", style = "heading 1") |>
  body_add_par("目的：...", style = "Normal") |>
  
  # 引言
  body_add_par("1 引言", style = "heading 1") |>
  body_add_par("研究背景...", style = "Normal") |>
  
  # 方法
  body_add_par("2 方法", style = "heading 1") |>
  body_add_par("2.1 研究设计", style = "heading 2") |>
  
  # 结果  
  body_add_par("3 结果", style = "heading 1") |>
  
  # 讨论
  body_add_par("4 讨论", style = "heading 1")

# 保存
print(doc, target = "07_paper/论文初稿.docx")
```
