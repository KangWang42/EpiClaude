# 真实论文内容范式（中文学位论文 / 中文期刊论著）

> 解决"写哪些内容、怎么写、像不像真论文"。本文件从真实学位论文（护理/公卫，照护负担类）反推出每一部分
> 到底该写什么、用什么句式、详略如何。**写中文学位论文/论著前必读**；尤其方法、讨论、结论最易写成
> "不像论文"的样子。配套 `chinese-paper.md`（结构与字数）、`review-killers.md`（硬伤）。

## 目录
1. 整体章节骨架
2. 摘要：四段怎么写
3. 研究对象与方法：写什么、详到什么程度
4. 统计分析方法：写成编号列表（最常写错）
5. 研究工具/量表：每个量表的固定写法
6. 结果：客观陈述 + 表图
7. 讨论：现状分析 + 相关性 + 影响因素逐个展开 + 质性 + 创新 + 局限
8. 结论：逐条定性，无统计量
9. 一句话对照：合格论文 vs 模板化稿件

## 1 整体章节骨架（学位论文）

```
1 前言
  1.1 研究背景
  1.2 国内外研究现状（1.2.1 国外 / 1.2.2 国内）
  1.3 研究目的及意义（1.3.1 目的 / 1.3.2 意义）
2 研究对象与方法
  2.1 量性研究：2.1.1 研究对象（纳入标准/排除标准/样本量计算）；
      2.1.2 研究方法（抽样方法/研究工具/资料收集方法/统计分析方法）
  2.2 质性研究：研究对象 / 研究方法（抽样/工具/资料收集/资料整理与分析）/ 质量控制 / 伦理原则 / 技术路线
3 结果
4 讨论（现状→相关性→影响因素逐个→质性体验→创新性→局限性）
5 结论（逐条）
参考文献 / 附录（量表）
```

期刊论著压缩版：前言 → 对象与方法 → 结果 → 讨论 → 结论，层级到 2 级为主。

## 2 摘要：四段

- **目的**：1–2 句，"调查……现状，探讨……影响因素，（质性）探索……真实感受，为……提供依据"。
- **方法**：写实打实的执行——研究设计 + **抽样方法 + 起止年月 + 地点** + 用到的**各量表** + **样本量** +
  软件 + **方法列表**（"主要统计学方法有描述性分析、t 检验、方差分析、Kruskal-Wallis 检验、Pearson/
  Spearman 相关、多重线性回归；访谈用 Colaizzi 七步法"）。
- **结果**：分条（1）（2）（3）……每条给**关键数字**（得分±SD、构成比、相关方向、回归保留的因素、质性主题）。
- **结论**：分条（1）（2）……**文字定性概括**，**不含 β/P/CI**，不复制结果细节。

字数 200–500（中文）。

---

## 3 研究对象与方法：写什么

- **研究对象**：目标人群 + 地点 + 时间；**纳入标准**与**排除标准**分条列（操作化、可复现，如"年龄≥18 岁；
  照顾年限≥半年、每日照顾≥4 小时；自愿并签知情同意"）；**样本量计算**给公式/依据。
- **抽样方法**：整群随机/目的抽样等，一句话。
- **资料收集方法**：谁、在哪、怎么收、用时多少（如"由普查员陪同入户一对一问卷，约 30 min"）。
- 量性与质性**分开各写一套**对象与方法。质性还要写**资料整理与分析**（转录时限、双人核对、编号建档、
  Colaizzi 七步）、**质量控制**（设计/调查/分析三阶段）、**伦理原则**（自愿/保密/有益）。

详略原则：方法要"够别人复现"，但**只写做了什么**，不写为什么这么选、不写方法学定义（见 §4、§9）。

## 4 统计分析方法：写成编号列表（最常写错的一节）

真论文的统计分析是一份**标准化清单**，每条"哪类资料 → 用哪种方法（→ 分析什么）"，一句一条，**零辩解、
零定义、零构念论证、零'为避免……偏倚'**。范式（照搬这个骨架，按你的方法替换）：

```
数据以 [软件版本] 分析，以 P<0.05 为差异有统计学意义。
（1）统计描述：计数资料以频数（构成比）描述；正态计量资料以均数±标准差描述，偏态以中位数（四分位间距）描述。
（2）单因素分析：正态且方差齐用 t 检验/单因素方差分析，否则用 Kruskal-Wallis 检验；分类变量间用卡方检验。
（3）相关分析：用 Pearson/Spearman 相关分析 X、Y 与结局的关系。
（4）多因素分析：以 [结局] 为因变量、[自变量集] 为自变量，作 [多重线性回归/逐步回归/因子回归]。
（5）质性资料采用 Colaizzi 七步分析法。
```

**不该进统计分析节的东西**（出现即改）：
- "采用合成得分的原因是……分布偏态/二分缺乏方差" → 删（动机辩解）。
- "X 是一种……的降维方法，将类别投影到……" → 删（教科书定义）。
- "不依据单因素 P 值取舍以避免逐步选择偏倚""维度间正交从构造上避免共线性" → 删（方法论辩护）。
- "分析前作数据质量核查/单维性核查" → 删（过程流水账）。
- **得分/指标的构建（赋分相加、维度命名、计分方式）属"研究工具/变量定义"，不放统计分析**。

诊断只在决定"报告哪个估计"时留半句（如"因检出异方差，回归系数区间以 Bootstrap 估计"）。

## 5 研究工具/量表：每个量表一个小节，固定五要素

```
该量表由 [作者] 于 [年] 提出，[中文版译者]，Cronbach α=[值]，信度良好[文献]。
共 [N] 个条目，分 [M] 个维度（……）。采用 [Likert N 级] 计分，[计分规则]，总分 [范围]，得分越[高/低]表示……。
依据得分分为：[分级界值与各级名称]。
```

自构问卷（如可及性问卷）：写清来源（自行设计）、组成部分、各条目计分与合成方式（**得分构建写在这里**，
不在统计分析节）。

## 6 结果：客观陈述 + 表图（分条要点式，不是数字流水账）

- 按研究问题/小节组织，每段先点表图（"见表 N""如图 N 所示"），再陈述**关键数字**，不复述整表。
- **纯客观，不解释原因**（原因留讨论）；分级、构成比、得分、相关方向、回归保留因素都给数字。
- 表注规范（检验方法、显著性标注）。

### 6.1 样本描述首段（清晰、结构化、信息完整）

研究对象/基线结果首段照此骨架写实写全：纳入总数 + 随访时长 + 各次测量 → 结局分组人数（事件 N / 非事件 N，
口径）→ "两组基本特征见表 N" → **逐变量点明哪些有/无统计学差异**（"年龄、性别、……在两组间均无统计学差异，
[关键基线变量] 有统计学差异"）。范式：

> 在 [起止时间]，本研究共纳入 [N] 名 [对象]，随访 [时长]，采集 [各次测量]。经 [随访] 后，[事件组 N] 例
> [结局] 进展（[操作化定义]），[非事件组 N] 例未进展。两组基本特征见表 N。结果显示，[变量列举] 在两组间
> 均无统计学差异，[关键变量] 有统计学差异（见表/附表 N）。

### 6.2 主结果用"结论先行 + 分条要点"，不是连续报数

主结果段**先给一句总判断**，再**分条逐点**说明，每条 = 一个子主题 + 该子主题下的发现（含关键数字佐证），
一条说清一件事。**严禁把多组数字连续罗列而不点出每组说明什么**。范式：

> 结果显示，本研究构建的模型对 [结局] 具有较高的预测价值。（1）[子主题一，如 时序/特征效果]：[一句话发现]，
> [关键数字佐证]。（2）[子主题二，如 不同组合的比较]：[发现]，[数字]。（3）[子主题三，如 某特征的增益]：[发现]，
> [效应量 + 95%CI]。（4）[子主题四，如 方法间比较]：[发现]。

要点：每条**先讲发现/结论**（这组对比说明什么），再用数字支撑；分条编号让结果层次清晰；差结果照报但同样分条说清。
对照同领域优秀学位论文的结果写法——分条、有重点、清晰，一个个说明，而非一段数字流。

## 7 讨论：五块，影响因素必须逐个展开

讨论不是"主要发现+笼统比较"，而是分块、且**影响因素逐因素成小节**：

1. **一般资料/现状分析**：给本研究数值与分级 → 与具体文献比较（"与 X 等[N] 接近/本研究高于城市") →
   **可能原因（1）（2）（3）**（结合该人群具体情境，如农村经济、外出务工、医疗资源）。
2. **相关性分析**：每对关系一段——"X 相关分析显示 A 与 B 呈负相关，即……越高、……越轻，与 X 等[N] 一致。
   这可能是因为……"。
3. **影响因素分析**：**每个显著因素一个小节**（4.4.1、4.4.2……），每节固定套路：
   ```
   单因素和多因素分析结果显示，[因素]越[X]，[结局]越[Y]，差异有统计学意义（P<0.05），（相关系数/β 为 Z），
   与 [作者] 等[N] 的研究结果一致/相似。这可能是因为 [结合该人群情境的机制解释，可分点]。
   ```
   这是讨论的主体；不要把多个因素揉成一段笼统说。
4. **质性体验**：按主题逐条讨论照护者真实体验与需求，与量性互证。
5. **创新性 / 局限性**：各成节；局限每条=局限点+影响+（可能的）应对。

机制解释用"这可能是因为""可能原因为"，保持审慎；横断面不下因果。

## 8 结论：逐条定性，无统计量

```
（1）[结局] 处于 [中度负担] 水平，[社会支持] 处于 [中等] 水平……（定性概括，无数字）
（2）[因素A、因素B……] 是影响 [结局] 的因素。
（3）[X 与 Y 显著相关，X 对 Y 有负向预测作用]。
（4）[质性结论 + 实践含义一句]。
```

**结论不出现 β/P/CI**，不复制结果细节，不与摘要逐字重复（各有侧重），不据探索性结果提具体分层方案
（见 review-killers §2）。

### 8.1 方法学/预测类研究的结论范式（贡献先行，正面收口）

调查/影响因素类用上面的逐条定性；**方法学、预测建模、技术框架类研究**照此正面收口——先讲"建立了什么"，
再讲"结果表明能带来什么、印证什么、提供什么依据"：

> 本研究提出了一种基于 [核心方法] 的 [框架/模型]，建立了能够 [核心能力] 的 [模型]，并进一步从 [角度一、角度二、
> 角度三……] 多个角度进行探索，为 [应用目标] 提供了可行路径。研究结果表明，[核心做法] 能够帮助 [使用者] 在
> [场景] 中 [实现什么]，印证了 [某做法] 的合理性与必要性，为 [某方向] 提供了方法学依据。

语言直白、清晰、客观、有效，像可发表论文的收口，不堆形容词、不含统计量。

### 8.2 结论硬红线：局限与不足绝不写进结论

- 结论**只写贡献、发现与意义**，是为发表服务的正面陈述；**严禁出现"受小样本约束""EPV 偏低""乐观偏差""泛化
  能力可能不足""真实性能预期低于报告值"等局限/自我削弱措辞**。
- 研究局限、caveat、外部验证需求**一律且只写在讨论的"研究不足"小节**（§7 第 5 块）；**结论与摘要结论里不写
  "尚需更大规模/多中心验证""结果有待进一步证实"这类验证展望句**——用户明确要求结论是纯正面贡献收口，
  外部验证需求只在讨论局限交代，不进结论/摘要结论。
- 把诚实的方法学边界写在讨论局限、把贡献与意义写在结论——二者分工明确不可混。真实的开发阶段乐观、口径偏差
  记在讨论局限与内部决策文档，不进结论/摘要结论。

## 9 一句话对照：合格论文 vs 模板化稿件

| 部位 | 模板化稿件（错） | 合格论文（对） |
|------|--------------|--------------|
| 统计分析 | 一段流水账 + 为什么这么选 + 方法是什么 + 避免什么偏倚 | 编号清单，每条"哪类资料→哪种方法" |
| 得分构建 | 塞进统计分析节并论证构念 | 写在研究工具/变量定义节 |
| 讨论影响因素 | "资源充裕与政策认知是保护因素"一句带过 | 每个因素一小节：结果方向+显著+文献+机制 |
| 结论 | 罗列 β=-5.11、P=0.001 | "X、Y 是影响负担的因素"定性逐条 |
| 语气 | 论证、辩解、自夸"先进/全面系统" | 平实陈述"做了什么、得到什么" |

## 10 前言与方法的写作逻辑（预测建模/流行病学类，细则）

> 以下是从优秀学位论文反推的**写作思路与逻辑**，按自己研究的内容套用——**学逻辑，不照抄结构**：每篇研究的疾病、
> 方法、设计不同，章节骨架与详写重点随之不同，不把别人的具体框架（某种分解方法、某种测量方案）硬搬过来。

### 10.1 主语原则（最易犯，全文通用）

**每句都要有明确主语**：本研究 / 课题组 / 患者 / 医师 / 研究者 / 模型 / 该队列 / 编码器。不要通篇无主语被动堆砌。
- BAD（无主语）："纳入完成多次随访的受检者。从每次随访记录提取指标。采用交叉验证评估。"
- GOOD（有主语）："本研究纳入完成多次随访的受检者；本研究从每次随访记录中提取……指标；并采用受检者分组
  交叉验证评估其判别力。"
- 方法每个操作句点明谁做："患者采取坐位……""医师用……仪器采集……""本研究将……纳入……模型"。
- **可执行自查（最易漏，必跑）**：`grep -nE "(^|。|；|后，|时，)(采用|计算|提取|纳入|检索|参照|评估|拟合|绘制|反推|重新计算)" 方法.md 结果.md`——句号/分句首即操作动词＝高度疑似缺主语。逐句确认：是真缺主语（补"本研究/研究者/结果"）还是名词性假阳性（"纳入标准""纳入的 N 篇""纳入综述的质量"——主语是名词短语，放过）。**人工读不可靠，这一类必须 grep 兜底**。
- 补主语要**变换**，不是每句都"本研究"：方法子节交替用"本研究 / 两名研究者 / 研究者 / 结果显示 / 该队列"；相邻子节不连续同一主语开头。

**不教科书式定义本研究的研究类型/方法**（和"无主语"同属方法部分的常见错误）：

- BAD（教科书式自我定义）："本研究为伞状综述（umbrella review），即对已发表的系统综述与 Meta 分析进行二次收集、统一再分析与证据强度分级。"——"X 是一种……，即对……进行……"是给读者讲概念，不是介绍自己做了什么。
- GOOD（直接说做了什么）："本研究按 Ioannidis 等确立的伞状综述评价框架[N]，对……各结局已报告的关联进行系统再评估，重新估计效应量、异质性与潜在偏倚并据预设标准分级。"
- 自查：`grep -nE "本(研究|文|综述)(为|是)[^，。]{0,25}[，,]即"` 命中＝删定义、改"本研究按/采用……对……进行……"。方法名、设计名直接用，读者自会查定义。

### 10.2 前言四节的逻辑

- **研究背景**：疾病定义（首次给"中文（English, ABBR）"）+ 流行/患病数据（**带文献**）→ 发病机制与不良后果
  → 早期识别/干预的意义（"因此，在……前期发现高危人群、尽早干预……具有公共卫生意义"）→ 现行金标准检查
  及其局限（成本、可及性）→ 替代生理信号手段的优势（价廉、无创、可重复）→ 自然引出研究空白。**每个关键医学
  概念首次出现都给英文全称 + 缩写**（疾病、检查手段、指标、调查名等，不止主结局）。
- **研究现况**：先点空白（"现有研究多为……，尚无针对……的研究"）→ 必要时用"第一步……；第二步……；最后一步
  ……"梳理该领域的通用分析流程（清晰且学术）→ **逐篇列代表性前人研究**，每篇一句到一段：[作者] 等[N] 用什么
  方法、纳入什么模型、达到什么**具体结果**（准确率/灵敏度/AUC 等数字）。让读者看到"别人做到哪、还差什么"。
- **当前研究存在的问题**：编号 (1)(2)(3)(4)，每条 = 问题陈述 → 为什么存在/有何危害 → 该方向的 gap → "因此，
  亟需/需要……"。这些问题逐条与研究目的呼应（提出的问题正是本研究要解决的）。
- **研究目的**：用**一句（或一段）有主语的完整目的句**收口——"本研究通过建立……队列，在……收集……，构建……，
  提取……，建立……模型，寻找/力图……，从而……"。目的句完整、有逻辑链、不分散，**不含任何局限或"尚需验证"**。

### 10.3 研究对象与方法的逻辑

- **研究对象**：起止时间 + 选取对象 + 队列建立与命名（中英全称 + 缩写）+ 伦理（遵循赫尔辛基宣言、伦理委员会
  批准 + 批号 `[待补充：批号]`）+ 知情同意 → **三级标准逐条编号**：`2.x.1 纳入标准：（1）…（2）…` /
  `2.x.2 排除标准：（1）…` / `2.x.3 剔除标准：（1）资料不全…（2）中途退出…`。每条操作化、可核查。
- **随访 / 数据采集**：写实写全且可复现——**具体设备型号**、操作步骤（有主语，如"患者采取坐位……医师粘贴电极……
