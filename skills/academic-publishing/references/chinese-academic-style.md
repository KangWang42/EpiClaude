# 中文学术写作文风规范（期刊论著 / 学位论文通用）

> 管"文风本身"：严肃到什么程度、用什么表达、什么句子、段落怎么组织、该写什么不该写。是**正向**风格指南。
> 分工：本文件给"该怎么写"的正面标尺；`chinese-style-audit.md` 给"模板化表达 + grep 审计线索 + 改写检查"（负向
> 清除）；`section-content-playbook.md` 给"各部件具体写什么内容"。三者配合：先按本文件定调写，再过 style-audit
> 扫痕迹，内容详略查 playbook。期刊与学位论文文风**同源**，差异见 §7。

## 1 总基调：严肃度标定

学术中文的恰当位置是**准确、克制、朴实**——介于"随意口语"与"繁复文言腔"之间，两头都不要：

- **不随意**：不用口语词（"搞""弄""挺""好多""差不多"）、网络语、感叹、反问、第一人称情绪（"我们惊讶地
  发现"）、对话腔（"那么""接下来我们看看"）。
- **不繁复**：不堆四字格与排比、不套文言腔、不写无信息的修饰（"众所周知""具有极其重要的意义和价值""在
  当今社会"）、不为显专业而叠长定语让句子难读。
- **落点**：像一位严谨研究者平实陈述"做了什么、得到什么、说明什么"。**信息密度高、修辞低**。能用一句平直
  陈述说清的，不绕。

一句话标尺：**读起来像可靠的同行评议论文，不像演讲稿，也不像散文**。

## 2 视角与人称

- 研究者视角，主语多为"本研究 / 本文 / 研究者"或客观主语（"结果显示""模型在……上""该队列……"）。
- 不用"我 / 我们认为 / 大家"；不用第二人称"你"；不用助手口吻（"建议把……""我们可以看到""值得注意的是"）。
- **每句都要有明确主语，不通篇无主语被动堆砌**（本研究 / 患者 / 医师 / 课题组 / 模型 / 该队列）。方法每个操作句
  点明谁做（"患者采取坐位……""本研究将……纳入……模型"）。详见 `section-content-playbook.md` §10.1。
  **这一项人工读极易漏（"检索……""采用……""参照……将……"读着顺就放过了），必须 grep 兜底**：
  `grep -nE "(^|。|；|后，|时，)(采用|计算|提取|纳入|检索|参照|评估|拟合|绘制|反推|重新计算)" 方法.md 结果.md`，命中句逐一确认主语。
- 陈述自己做的事用主动："本研究纳入……""采用……模型"；不滥用被动到读不出谁做的。
- 报告他人工作用"……等[N] 发现/报告"，立场审慎。

## 3 句子层面

- **句式服从清晰度**：短句、中句为主，复杂关系才用长句；避免连续复句堆叠，不设置机械句长配额。
- **一句一个意思**：避免一句里塞多个并列论断；复杂关系拆成两三句。
- **主谓宾清晰**：避免多层定语套叠（"……的……的……的……"）、避免超长前置状语把主语推到句尾。
- **客观陈述句为主**：结果章纯陈述事实与数字，不在句中夹解释/评价（解释留讨论）。
- **数字与术语**：术语首次出现给全称 + 缩写，此后统一用缩写；数字精度、单位、CI/P 写法全文一致；量值
  紧跟其指代对象，不悬空。
- **慎用**：慎用"显著"（仅指统计学显著、与 P 对应，不作"明显"用）；慎用绝对化（"完全""彻底""必然"）；
  观察性研究不用因果动词（"导致""引起""证明"），改"与……相关""提示"。

## 4 段落层面

- **一段一个论点**：段首一句主题句点明本段要说什么，后续句围绕它展开（数据/对照/解释），不跑题、不并置无关点。
- **逻辑衔接靠语义，不靠模板**：避免"首先/其次/再次/最后""一方面/另一方面"通篇套用；连接词收敛、必要时才用，
  靠句子内在逻辑承接。
- **段落长度适中**：一般 3–6 句；过长（半页一段）拆分，过碎（一句一段）合并。
- **结果段范式**：先点表图（"见表 N""如图 N 所示"）→ 再述关键数字 → 不复述整表。
- **讨论段范式**：本研究发现 → 与具体文献对照（高于/接近/低于谁）→ 审慎机制解释（"可能的原因是"）→ 不外推。
- 段与段之间有递进/转折的实际逻辑，不是平行罗列；章节开头可用一两句承上启下，但不写空话过渡。

## 5 该写 vs 不该写（速查）

| 维度 | 该写 | 不该写 |
|---|---|---|
| 方法 | 做了什么、够别人复现 | 方法学定义、教科书式介绍、"为避免……偏倚"的辩解 |
| 统计 | 编号清单"哪类资料→哪种方法" | 一段流水账 + 为什么这么选 + 构念论证 |
| 结果 | 客观数字、方向、差结果照报 | 原因解释、自我评价、删差结果、防误读的"需指出…受…影响…而非…"类限定句（属讨论/局限） |
| 讨论 | 文献对照 + 审慎机制 + 意义 + 不足 | 重复结果细节、笼统比较、自夸"先进/全面系统" |
| 结论 | 逐条定性、落到意义 | β/P/CI、复制结果、据探索性结果提具体方案 |
| 语气 | 平实陈述 | 论证腔、辩解、宣传腔、情感词 |
| 修饰 | 必要、具体 | "众所周知""极其重要""具有重要意义和价值"等空话 |

- **正文只写做了什么，不写没做什么**：方法/结果**NEVER** 出现"未设 / 未做 / 没有 / 未纳入 X"的自陈句——没做的、缺的统一进讨论的**局限节**；正文措辞中性、既不 claim 没做到的（如未做独立外部验证，就既不写"独立/外部验证"、也不在正文写"未设独立外部验证"）。
- **图/表题简洁，符合论文规范**：图题=图号+名词短语标题（必要时括注图内元素含义，如"格内为分期、例数与5年OS"、误差条含义、缩写）；**NEVER** 把图的版面布局/构造逻辑全写进图题（"左为…右为…由…合并…"）——那些靠图本身展示、或已在正文交代。图内不嵌入图题/副标题（标题走 docx 图注，见 publication-figures §1）。

（"什么内容进哪一节"的细则见 `section-content-playbook.md`；模板化表达审校见 `chinese-style-audit.md`。）

## 6 用词

- 用领域内**具体、低频的专业词**而非泛词（"回归均值"而非"一种现象"；"糖化血红蛋白（HbA1c）"而非"某个血糖指标"）。
- 同一概念全文用同一词，不换着叫（变量名/模型名/构念名前后一致）。
- 方法名称、审校步骤和图内标签使用原文、研究方案、领域指南或可核验方法文献中的规范术语。不得把内部流程词、软件工程隐喻、生硬直译或临时拼接的四字短语写入正文；不能确认时保留原词并标记待确认。
- 不用宣传词（"创新性地""突破性""填补空白""首次提出"需有据，无据删）、不用模糊归因（"有研究表明"要
  给具体文献）、不用表情符号 / 破折号卖弄 / 粗体滥用。
- 中英混排：英文缩写、外文专有名词按需保留，句子主体是规范中文；标点用中文全角。
- **连接用规范书面语，不用半文不白的"辅以 / 佐以 / 兼以"**：直接写"以 X 做 Y""采用 X 进行 Y""并用 X 校正"，不写"辅以 X 进行 Y"。
- **专业术语首次出现给"中文全称（English full name, ABBR）"，此后统一用缩写**，且**贯穿全文每个关键概念**
  （不止主结局/主方法）：如"高血压（hypertension, HTN）""心血管疾病（cardiovascular disease, CVD）""左心室
  肥厚（left ventricular hypertrophy, LVH）"。只给主结局加全称、其余概念裸用中文或裸用缩写都算不规范。
- **原始数据标签 / 编码不照搬进正文散文**：年龄组标签（"0—""85及以上"）、变量类别码、列名、单位后缀等是数据表里
  的机读写法，正文须转成自然学术语言（"0—、1—……9— 岁的单岁组"→"0—9 岁按单岁分组"）。表格与图轴可保留这些标签，
  散文不可照搬——直搬即读着别扭、暴露数据处理痕迹。
- **同一事实只在最合适处交代一次**：地理单元、分组口径、样本量等不在方法多个小节重复陈述（如 2.1 已说"按地区与
  年龄组分层"，2.2 不再罗列各组）；后文需要时引用，不复述。

## 6bis 严禁的随意 / 不负责任表达（语言硬红线，期刊与学位通用）

以下表达不符合学术语体。命中即改，交付前 grep 自查归零：

- **设问句 / 反问句**：正文严禁用问句制造悬念或过渡，如"……能否……？""这是否意味着……？""为什么会这样？"。
  一律改为陈述句（"上述局限提示，需要一种……的手段"而非"……能否……？"）。学术正文几乎不应出现"？"。
- **口语化随意断言**："最关键现象出现在……""乍看……实际上……""有意思的是""说白了""其实"——一律删或改为
  客观陈述。结果的轻重由数字与统计量说话，不用主观形容词替读者下"最关键/最重要"的判断。
- **对话残留 / 元叙述**："值得指出的是""需要指出的是""不难发现""我们可以看到""接下来""下面""如前所述"
  （除非确为必要的交叉指引）——删。论文是客观陈述，不是讲解。
- **空泛评价开头**：避免"对……的早期识别不仅关系到……还直接影响……""……具有重要的意义和价值""……
  显得尤为重要"这类无信息的价值判断式开头（见 §4 段落开头）。
- **不负责任的程度词**：无统计支撑的"显著上升/急剧增加/大幅"、无依据的"众所周知/普遍认为"——删或补依据。

## 6ter 段落开头句式（避免套路化）

每段段首是主题句，直接进入本段事实，**避免固定评价式套路开头**。

| 段落类型 | BAD（套路/评价式开头） | GOOD（事实/主题句开头） |
|---|---|---|
| 背景 | 众所周知，X 是当今社会重要的问题…… | X 以……为主要特征，其患病率…… / 据某调查，X 患病率为…… |
| 意义过渡 | 对这两类损害的早期识别不仅关系到……还直接影响…… | 在基层随访场景中，标准检查受……限制，难以…… |
| 结果 | 最关键的现象出现在…… | 表 N 显示，事件组与非事件组在……上的差异为…… |
| 方法 | 为了科学严谨地…… | 本研究采用……，按……执行。 |

开头多样化：陈述对象、给数据、点表图、述操作交替使用，不要每段都"对……来说"/"值得注意"。

**研究目的段不用轻连接词起头**（引言收口最常见）：

- BAD："基于此，本研究对……开展……"／"综上，本研究……"——"基于此/综上/由此/据此"是空转连接词，把目的句接得短而轻，读着不像论文收口。
- GOOD：用实义过渡承接前文点出的空白——"针对上述证据分散、强度高低难辨的现状，本研究对……开展……，以……"。让目的句与前一段的 gap 形成因果，而非靠"基于此"贴上去。
- 自查：`grep -nE "^(基于此|综上|由此|据此|因此)[，,]本(研究|文)"` 命中即改。

## 7 期刊论著 vs 学位论文的文风差异

文风同源（都严肃克制朴实），差在**展开度**：

| 维度 | 期刊论著 | 学位论文 |
|---|---|---|
| 详略 | 紧凑、惜字、每句承重 | 可更展开、交代更完整（方法步骤、背景脉络写全） |
| 现状/综述 | 引言里几句点空白 | 前言现状成节 + **独立综述章**系统述评 |
| 解释 | 点到为止 | 机制/对照可逐点更充分，但仍审慎不注水 |
| 自我定位 | 一句创新点 | 研究特色可成子节，但用事实说话不自夸 |
| 篇幅压力 | 控字数、删冗余 | 达字数下限靠"写全该写的"，**不靠注水堆砌** |

学位论文"长"不等于"啰嗦"：每一节仍要信息承重，扩的是覆盖面（更多角度、更全交代），不是水词。

## 8 GOOD / BAD 微例

聚焦文风（审计模式另见 `chinese-style-audit.md`）：

| 场景 | BAD（随意/繁复/论证腔） | GOOD（平实承重） |
|---|---|---|
| 开头 | 众所周知，高血压是当今社会极其重要的公共卫生问题，具有重大意义。 | 高血压是心血管疾病的主要危险因素，其靶器官损害的早期识别仍存在不足。 |
| 方法 | 为了科学严谨地避免选择偏倚，我们精心采用了先进的交叉验证方法。 | 采用受检者分组 5 折交叉验证，按受检者分折以避免同人数据跨折泄漏。 |
| 结果 | 我们惊喜地发现模型表现非常优异，效果好得令人满意。 | 纳入基线人口学与实验室指标的预测模型在验证集的 AUC 为 0.78（95% CI 0.72–0.84）。 |
| 讨论 | 这一结果充分证明了本方法的巨大优势和广阔前景。 | 该结果与既往报告方向一致；其关联可能与慢性炎症水平升高有关。 |
| 句式 | 由于在考虑了多种因素并经过反复权衡之后最终我们决定采用的这一方法…… | 综合可解释性与样本量限制，主分析采用该方法。 |
| 口径 | 年龄分层为 0—、1—……9— 岁的单岁组与 10—、15—…… | 0—9 岁按单岁分组，10 岁及以上按 5 岁分组至 85 岁及以上。 |

## 9 文风自检清单

- [ ] 通篇学术书面语：无口语/网络词/感叹反问/对话腔/第一人称情绪
- [ ] 无 §6bis 随意/不负责任表达（值得指出的是/最关键现象出现在/乍看…实际上/不难发现…全部归零）
- [ ] 每个关键术语首次出现给"中文全称（English, ABBR）"，全文一致用缩写
- [ ] 段首无套路化评价式开头（对…不仅…还…/众所周知/值得注意），主题句直入事实
- [ ] 无空话修饰与宣传腔（众所周知/极其重要/具有重要意义/创新性地/填补空白…无据即删）
- [ ] 方法/结果每个操作句有明确主语（grep 句首操作动词兜底，见 §2），且主语变换不机械"本研究"
- [ ] 不教科书式定义本研究的研究类型/方法（无"本研究为X，即对…进行…"）；研究目的段不用"基于此/综上"轻连接开头
- [ ] 句长有变化、一句一意、无多层定语套叠；关键结论用短句
- [ ] 段落一段一论点、段首主题句、衔接靠语义不靠"首先其次"模板
- [ ] "显著"仅对应统计学显著；观察性不用因果词；无绝对化措辞
- [ ] 术语首次全称、全文同一概念同一词、数字精度统一
- [ ] 方法与图内标签均可追溯到来源或规范术语，无自造缩略语、复合词、动作式口语或内部流程词
- [ ] 正文散文无原始数据标签/编码直搬（年龄"0—"、类别码、列名、单位后缀转自然语言；表图可留）；同一事实不在多处重复交代
- [ ] 结果只陈述不解释、差结果照报；讨论对照具体文献 + 审慎机制
- [ ] 学位论文：展开但不注水，每节信息承重
- [ ] 已过 `chinese-style-audit.md` 的不可变事实清单、论断强度与模板化表达校验
